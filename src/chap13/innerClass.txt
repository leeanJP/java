내부 클래스
 '클래스 내부에 선언한 클래스'
 class Out{       //외부 클래스
    class In{     //인스턴스 내부 클래스

    }
    static class Sin { // 정적 내부클래스

    }

    public void abcd(){
        class Local{  //지역 내부 클래스
        }
    }
 }

선언하는 위치나 예약어에 따라서
-인스턴스 내부 클래스
-정적 내부 클래스
-지역 내부 클래스
-익명 내부 클래스
내부 클래스의 종류에 따라서 만드는 방법도 다르고
클래스 내부에 선언할 수 있는 변수 유형과
사용할 수 있는 외부 클래스 변수 유형도 다르다.


*인스턴스 내부 클래스
*익명 내부클래스
- 이름이 없는 클래스
- 변수에 직접 대입하는 경우도 있고
  메서드 내부에서 인터페이스나 추상클래스를 구현하는 경우도 있다.
